# Vastu Gurukul - Landing Page

A single-page explanation site for Vastu Gurukul, the Astro Vastu and MahaVastu brand founded and led by Acharya Monica.
Static HTML/CSS/JS - no build step, no dependencies.

## Run it
```bash
cd "C:\Users\Abhay\OneDrive\Desktop\MONICA MAHAVASTU"
python3 -m http.server 8000
```
Then open **http://localhost:8000** in your browser.

(Any static server works - e.g. `npx serve` if you have Node.)

## What's inside
- `index.html` - the entire site (styles and scripts are inline)
- `images/monica-hero.png` - the real photo of Monica used in the hero and about sections
- `CLAUDE.md` - full project context for Claude Code; read it first
